前言

上一篇文章为大家介绍了如何配置GitHub、安装Hexo框架以及Hexo的几个基本命令。经过上篇文章的准备后，我们博客的雏形已经搭建成功了，若相关准备尚未完成，请自行翻看前面的文章。如果对前面介绍的Hexo命令还没完全掌握，请不用着急，因为这几个命令在之后博客的配置和使用中还会反复用到，到时我还会再次给大家讲述各个命令的作用。

本篇文章将为大家介绍如何为自己的Hexo博客更换NexT主题以及对NexT主题进行优化配置，从而让我们的博客外观更加好看。本篇的主要内容将包括：

  • 下载及启用NexT主题
  • NexT主题美化
    • 选择你喜欢的主题框架（scheme）
    • 在侧边菜单栏中添加“标签”页面
    • 侧边菜单栏图标设置
    • 在侧边菜单栏中增加社交链接按钮
    • 开启背景特效
  • 博客功能扩展
    • 增加第三方服务“百度统计”
